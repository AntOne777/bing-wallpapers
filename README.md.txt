# Горизонт

Личный архив визуальной истории Bing.

## О проекте
Архив ежедневно собирает и сохраняет фоновые изображения Bing из разных регионов. Проект спроектирован как полностью автономная система, не требующая базы данных или серверной части.

## Архитектура
- **Collector (`fetch_bing.py`):** Python-скрипт. Ежедневно опрашивает Bing API для 11 регионов. Работает через GitHub Actions.
- **Storage (`data.json`):** Единый реестр всех изображений с метаданными (ID, название, дата, доступные рынки).
- **Viewer (`index.html`):** Статический интерфейс. Реализует lazy-loading, управление памятью через Blob и экспорт оригиналов в UHD.

## Ключевые особенности
- **Дедупликация:** Изображения сохраняются по уникальному `img_id`.
- **История:** Сохраняется список всех регионов, где картинка была актуальна.
- **Архивный экспорт:** Файлы сохраняются с именами вида `ID_YYYYMMDD.jpg`.
- **Стабильность:** Код защищен от XSS, ошибок загрузки и утечек памяти (revokeObjectURL).

## Как просматривать локально
Поскольку проект использует AJAX-запросы к `data.json`, браузеры блокируют открытие `index.html` напрямую через файловую систему. Чтобы запустить локально:
1. Открой терминал в папке проекта.
2. Выполни команду: `python -m http.server 8000`
3. Перейди по адресу: `http://localhost:8000`
