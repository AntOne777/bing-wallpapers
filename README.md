Горизонт
Это мой личный архив фоновых изображений от Bing.

Зачем я это сделал?
Мне всегда нравились обои Bing, но они пропадают уже через сутки. Я решил, что нужно место, где они будут копиться и храниться постоянно. Так появился «Горизонт» — это не просто папка с картинками, а автоматизированный архив, который сам забирает свежие обои из 11 разных регионов мира.

Как это работает
Сборщик (Collector): Я написал скрипт на Python, который каждые 2 часа проверяет Bing по разным странам. Все это крутится само по себе через GitHub Actions — мне не нужно даже открывать ноутбук.

База (Storage): Всё хранится максимально просто, без сложных баз данных. Есть data.json, где записано, когда и где картинка появилась, а сами обои скачиваются в исходном 4K разрешении.

Просмотрщик (Viewer): Для index.html я сделал простую галерею. Она быстро грузится, умеет открывать обои на весь экран и скачивать их в полном качестве. Память браузера не забивается, потому что я аккуратно чищу временные объекты.

Фишки архива
Никаких дублей: Если обои появились в США и, скажем, в Германии — скрипт просто добавит информацию об этом в JSON, а картинку второй раз качать не будет.

Имена файлов: Я сохраняю их ровно так, как дает Bing (например, AurayBrittany.jpg), чтобы всегда можно было понять, откуда файл пришел.

Надежность: Тут нет никаких скрытых зависимостей или левых API. Проект автономный, сломаться в нем особо нечему.

Как запустить у себя
Если хочешь посмотреть, как это выглядит локально:

Скачай проект.

В папке с проектом открой терминал и напиши: python -m http.server 8000.

Открой в браузере http://localhost:8000 — и готово.
