# Горизонт

Личный архив визуальной истории Bing.

## О проекте

«Горизонт» — автономная система сохранения и просмотра фоновых изображений Bing Wallpaper из разных регионов мира.

Архив ежедневно собирает новые изображения, сохраняет их оригиналы и накапливает историю распространения по рынкам. Проект не использует базу данных или серверную часть и полностью работает как статическая коллекция файлов.

## Архитектура

### Collector (`fetch_bing.py`)
Python-скрипт, отвечающий за сбор данных.

- Ежедневно получает актуальные изображения Bing;
- Опрашивает 11 регионов;
- Запускается автоматически через GitHub Actions;
- Выполняет проверку существующих записей.

### Storage (`data.json`)
Единый реестр архива.

Хранит метаданные изображений:
- уникальный ID;
- название;
- дату первого появления;
- URL оригинала;
- список регионов, где изображение было доступно.

### Viewer (`index.html`)
Статический интерфейс просмотра.

Возможности:
- lazy-loading изображений;
- полноэкранный просмотр;
- скачивание оригиналов UHD;
- управление памятью браузера через `Blob` и `revokeObjectURL`.

## Ключевые особенности

### Дедупликация
Каждое изображение сохраняется один раз по уникальному `img_id`. Повторные появления не создают дубликаты, а расширяют историю региональной доступности.

### История распространения
Архив сохраняет информацию о регионах, в которых изображение Bing было опубликовано. Таким образом создаётся не просто коллекция картинок, а временная карта распространения контента.

### Архивный формат
Файлы сохраняются в формате: `IMG_ID_YYYYMMDD.jpg`
Пример: `AurayBrittany_20260711.jpg`

### Надёжность
Проект рассчитан на долгосрочное хранение:
- отсутствие внешних зависимостей;
- защита от дубликатов;
- обработка ошибок загрузки;
- безопасное формирование имён файлов;
- отсутствие утечек памяти через `revokeObjectURL`.
